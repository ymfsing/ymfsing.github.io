我博客使用的主题是 hugo theme even ,不过默认不支持搜索，在 issues 289 中找到解决方法，记录一下。

hugo-search-fuse-js 插件

从 GitHub repo 下载到本地博客 theme 文件夹中，或者使用 git submodule 下载。

配置

config.toml 

1

theme = ["hugo-search-fuse-js", "my-theme"]

layout/_default/baseof.html

由于主题中没有定义 main 和 footer block, 需自己添加。
